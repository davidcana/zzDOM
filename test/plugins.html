1. Hello, world!

zz( '#t1-1' ).hello();

white. Original value is white, modified must be Hello, world!.

zz( '#t1-2 .selected' ).hello();

  • white. Original value is white, modified must be Hello, world!.
  • black. Original value is black, must not be modified.
  • red. Original value is red, modified must be Hello, world!.

2. isNumeric

zz( '#t2-1' ).isNumeric();

  • white. Original value is white, must not me modified.
  • 7. Original value is 7, must not me modified.

zz( '#t2-2 .t2-2' ).isNumeric(); -> true

  • white. Original value is white, must not me modified.
  • 7. Original value is 7, must not me modified.

zz( '#t2-3 .t2-3' ).isNumeric(); -> false

  • white. Original value is white, must not me modified.
  • red. Original value is red, must not me modified.
