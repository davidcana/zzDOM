1. ZZDom selectors

zz( '#', 't1-1' ).text( 'yellow' );

white. Original value is white, modified must be yellow.

zz( '.', 't1-2' ).text( 'yellow' );

  • white. Original value is white, modified must be yellow.
  • red. Original value is red, modified must be yellow.

zz( 't', 'i' ).text( 'yellow' );

  • white. Original value is white, modified must not be changed.
  • red. Original value is red, modified must be yellow.
  • black. Original value is black, modified must be yellow.

zz( 'n', 'myname' ).text( 'yellow' );

  • white. Original value is white, modified must not be changed.
  • red. Original value is red, modified must be yellow.
  • black. Original value is black, modified must be yellow.

zz( 's', '.t1-5 span.selected' ).text( 'yellow' );

  • white. Original value is white, modified must not be changed.

  • black. Original value is black, modified must be yellow.
